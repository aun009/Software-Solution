# Software Store Theme Manifest

This document preserves the "Mission Control" Dark Theme and "Corporate Dashboard" Light Theme configurations used during development.

## 🔳 Mission Control (Dark Theme)
- **Primary Foundation:** `#050505` (Deep Charcoal)
- **Elevated Surfaces:** `#0a0a0a` & `#0d0d0d`
- **Glass Layers:** `rgba(255, 255, 255, 0.02)`
- **Atmospheric Gradients:** 
  - Deep Navy: `rgba(30, 58, 138, 0.1)`
  - Midnight Purple: `rgba(88, 28, 135, 0.1)`
- **Typography:**
  - Primary: `gray-100` (#F3F4F6)
  - Secondary: `gray-400` / `gray-500`
- **Accents:**
  - Blue: `blue-500` (#3b82f6)
  - Purple: `purple-500` (#a855f7)
  - Emerald: `emerald-400`
- **Textures:**
  - Dot-Grid: `rgba(255, 255, 255, 0.05)`

## ☀️ Corporate Dashboard (Light Theme)
- **Primary Foundation:** `#F8FAFC` (Slate-50)
- **Elevated Surfaces:** `#FFFFFF` (Pure White)
- **Glass Layers:** `rgba(255, 255, 255, 0.6)`
- **Atmospheric Gradients:**
  - Soft Sky: `rgba(59, 130, 246, 0.08)`
  - Soft Lilac: `rgba(168, 85, 247, 0.06)`
- **Typography:**
  - Primary: `gray-900` (#111827)
  - Secondary: `gray-600`
  - Micro: `gray-500`
- **Accents:**
  - Blue: `blue-600` (#2563eb)
  - Purple: `purple-600`
  - Emerald: `emerald-500`
- **Textures:**
  - Dot-Grid: `rgba(0, 0, 0, 0.04)`

## Global Assets
- **Sans Font:** Inter
- **Display Font:** Space Grotesk
- **Serif Font:** Playfair Display (Italic)
